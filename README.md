# AtliQ-Hospitality-Analytics-Dashboard
AtliQ Grands Hospitality Analysis is a Power BI project that analyzes hotel performance using KPIs like Revenue, RevPAR, ADR, Occupancy %, and Realization %. It provides city, property, and channel‑level insights to support data‑driven pricing and marketing decisions.


Objectives-

Analyze revenue performance across cities, properties, and booking channels.

Track key hotel KPIs: Revenue, RevPAR, ADR, Occupancy %, Realization %, Room Nights.

Identify high‑ and low‑performing channels and cities for better marketing and pricing.



Data & Tools-

Tool: Power BI Desktop

Domain: Hospitality (hotel chain – AtliQ Grands)

Source: Codebasics Hospitality Resume Project Challenge
